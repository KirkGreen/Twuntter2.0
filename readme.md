## Setup:
1. npm i
2. cd client 
3. npm i

## How to run:
  - start node server
    ```shell script
      npm start 
    ```
  - start nodemon server
    ```shell script
      npm run server
    ```
  - start client
    ```shell script
      npm run client 
    ```
  - start server and client
    ```shell script
      npm run dev 
    ```
